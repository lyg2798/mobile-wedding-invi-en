# Mobile Wedding Invitation — JQ & Erin

A calm, warm mobile wedding invitation for **JQ (YoungGyu) Lee & Erin Lee**, October 9, 2026 at The War Memorial of Korea, Seoul.

Built with Vite + React from a design handoff (Claude Design).

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Design notes

- Palette: warm cream / soft terracotta / muted sage / warm charcoal.
- Type: Cormorant Garamond (serif), Nunito (sans), Italiana (script).
- Sections: Sticky header → Hero → Invitation note → Calendar → Location → Footer.
- The hero photo lives at `public/couple-hero.jpg`.
